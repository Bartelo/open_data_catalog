Why reinvent the wheel, you might not be the only person wit this problem. Perhaps that same problem has already been addressed through an awesome locally built web app and we’d like to share it! Sharing the software you’re building is the best way to ensure that quality remains high and that other can benefit from and contribute to it.

If you have a potentially shareable application in development or production, let us know. If it’s not already released, we can help you develop plans to share it. If it’s already available, we can help publicize it by adding it to the Commons.
